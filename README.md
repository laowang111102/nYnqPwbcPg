# Python智能文献管理系统 python654

## 项目概述

本项目是一款基于Python语言及Django框架开发的智能文献管理系统。通过协同过滤算法和MySQL数据库技术，实现文献的高效管理与推荐。

## 技术栈

- 后端：Python Django
- 数据库：MySQL
- 推荐算法：协同过滤

## 功能模块

### 用户管理

- 用户注册、登录、密码找回等功能

### 文献类型管理

- 文献分类、类型维护

### 文献信息管理

- 文献上传、下载、预览、搜索、推荐

### 文献注释管理

- 用户对文献进行注释、查看他人注释

### 在线论坛

- 用户可发布帖子、回复、点赞、讨论

### 系统管理

- 系统设置、用户权限管理、日志管理等

### 我的信息

- 个人信息维护、我的收藏、我的足迹等

## 系统角色

- 普通用户：可使用文献搜索、下载、讨论等功能
- 管理员：拥有系统管理、用户管理等权限

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x

## 部署步骤

1. 安装Python环境和Django框架
2. 安装MySQL数据库，创建数据库和用户
3. 导入项目SQL文件，初始化数据库
4. 修改项目配置文件，配置数据库连接
5. 运行Django项目

## 目录结构

```
project/
├── apps/            # 业务模块
├── db/              # 数据库文件
├── media/           # 静态文件
├── migrations/      # 数据库迁移文件
├── project/         # 项目主目录
├── static/          # 静态资源
└── templates/       # 模板文件
```
